About WeatherExpress

Welcome to WeatherExpress

WeatherExpress is a user-friendly web application that allows you to easily get weather information for any city in the world. With a simple and intuitive interface, you can quickly search for a city, select your preferred temperature unit, and view the current weather conditions, including weather summary, weather description, and high/low temperatures for different times of the day.

Weather App also provides an interactive map powered by the Google Maps API, allowing you to visually explore and interact with the map to view weather information for different locations on the map.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Flask (Python web framework)
  • Google Maps JavaScript API (for interactive map)
